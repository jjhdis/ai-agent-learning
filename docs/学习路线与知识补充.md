## 2. 第一阶段：链式组合（Chain）

### 2.1 什么是 Chain？

你的代码目前是**一个独立的 Agent 循环**。Chain 的思想是：**把多个处理步骤串联成一个管道**，前一步的输出是后一步的输入。

### 2.2 当前代码的局限

```python
# 你的代码：只能做一件事
result = agent.run("上海天气")  # 一个 Agent 做完所有事
```

```python
# LangChain：可以组合多个步骤
chain = (
    {"context": retriever, "question": RunnablePassthrough()}  # 步骤1：检索
    | prompt_template                                           # 步骤2：组装提示词
    | llm                                                       # 步骤3：调 LLM
    | output_parser                                             # 步骤4：解析输出
)
result = chain.invoke("上海天气")
```

### 2.3 需要补充的知识

| 知识点 | 说明 | 实践建议 |
|---|---|---|
| **Runnable 协议** | LangChain 的统一接口，任何组件都实现 `.invoke()` / `.stream()` / `.batch()` | 给你的 Agent 也加上统一的 `invoke()` 方法 |
| **LCEL（LangChain Expression Language）** | 用 `\|` 符号组合多个 Runnable 的 DSL | 实现一个简单的 Pipeline 类，支持 `pipe()` 方法 |
| **Chain 类型** | `LLMChain`、`SequentialChain`、`RouterChain` 等 | 先实现一个 `SequentialChain`（顺序执行多个步骤） |
| **RunnablePassthrough / RunnableMap** | 数据传递和变换的工具 | 实现一个简单的数据传递中间件 |

### 2.4 在你的代码中实践

```python
# 你可以这样扩展：实现一个简单的 Pipeline
class Pipeline:
    """简单的链式组合，类似 LangChain 的 LCEL"""
    def __init__(self, steps: list):
        self.steps = steps
    
    def run(self, input_data):
        result = input_data
        for step in self.steps:
            result = step(result)
        return result

# 使用
pipeline = Pipeline([
    retrieve_documents,    # 先检索
    build_prompt,          # 再组装提示词d
    agent.run,             # 再调 Agent
])
```

## 3. 第二阶段：记忆系统（Memory）

### 3.1 当前代码的局限

| **ConversationSummaryMemory** | 用 LLM 总结对话，节省 Token | 实现一个自动摘要功能 |

```python
class ConversationBufferMemory:
    """简单的对话记忆"""
    def __init__(self, max_history: int = 10):
        self.history: list[dict] = []
        self.max_history = max_history
    
    def add_message(self, message: dict):
        self.history.append(message)
        if len(self.history) > self.max_history * 2:
            self.history = self.history[-self.max_history * 2:]
    
    def get_history(self) -> list[dict]:
        return self.history
    
    def clear(self):
        self.history.clear()

# 改造 Agent，支持外部记忆
class Agent:
    def __init__(self, ..., memory: ConversationBufferMemory = None):
        self.memory = memory or ConversationBufferMemory()
        self.history = self.memory.get_history()  # 从记忆恢复历史
```

```python
class CallbackManager:
    """管理多个回调处理器"""
    def __init__(self):
        self.handlers: list[BaseCallbackHandler] = []
    
    def add_handler(self, handler):
        self.handlers.append(handler)
    
    def on_event(self, event: str, *args, **kwargs):
        for handler in self.handlers:
            method = getattr(handler, f"on_{event}", None)
            if method:
                method(*args, **kwargs)
```

### 7.4 在你的代码中实践

```python
class Agent:
    def run_stream(self, user_message: str):
        """流式版本的 run"""
        self._emit("on_start", user_message)
        self.history.append({"role": "user", "content": user_message})
        tools = self.registry.get_openai_definitions()
        
        for step in range(self.max_iterations):
            msg = self.llm.chat(self.history, tools if tools else None)
            
            if msg.content:
                # 逐字符产出思考内容
                for char in msg.content:
                    yield ("think", char)
            
            if not msg.tool_calls:
                self.history.append({"role": "assistant", "content": msg.content})
                # 逐字符产出最终回复
                for char in msg.content:
                    yield ("reply", char)
                return
            
            self.history.append(msg.model_dump())
            self._handle_tool_calls(msg.tool_calls)
            yield ("tool_result", "工具执行完成")

# 使用
for event_type, content in agent.run_stream("上海天气"):
    if event_type == "reply":
        print(content, end="", flush=True)
```

### 9.4 Plan-and-Execute 简单实现

```python
class PlanAndExecuteAgent:
    """先规划，再执行"""
    def __init__(self, llm, registry):
        self.llm = llm
        self.registry = registry
    
    def run(self, task: str) -> str:
        # 阶段1：制定计划
        plan_prompt = f"为以下任务制定执行计划：{task}\n请列出步骤。"
        plan = self.llm.chat([{"role": "user", "content": plan_prompt}])
        
        # 阶段2：逐步执行
        results = []
        for step in plan.content.split("\n"):
            if step.strip():
                result = self.execute_step(step.strip())
                results.append(result)
        
        # 阶段3：汇总结果
        summary_prompt = f"基于以下执行结果，回答用户问题：{task}\n结果：{results}"
        summary = self.llm.chat([{"role": "user", "content": summary_prompt}])
        return summary.content
    
    def execute_step(self, step: str) -> str:
        # 用 ReAct Agent 执行单个步骤
        agent = Agent(llm=self.llm, registry=self.registry)
        return agent.run(step)
```

```python
import time
import functools

def retry(max_retries=3, delay=1.0):
    """重试装饰器"""
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kwargs):
            for i in range(max_retries):
                try:
                    return func(*args, **kwargs)
                except Exception as e:
                    if i == max_retries - 1:
                        raise
                    time.sleep(delay * (2 ** i))  # 指数退避
            return None
        return wrapper
    return decorator

class RateLimiter:
    """简单的速率限制器"""
    def __init__(self, max_calls: int, period: float):
        self.max_calls = max_calls
        self.period = period
        self.calls = []
    
    def wait_if_needed(self):
        now = time.time()
        self.calls = [t for t in self.calls if now - t < self.period]
        if len(self.calls) >= self.max_calls:
            sleep_time = self.calls[0] + self.period - now
            if sleep_time > 0:
                time.sleep(sleep_time)
        self.calls.append(time.time())
```
